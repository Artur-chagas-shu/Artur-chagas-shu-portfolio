Sobre mim

Desenvolvedor backend em formação com foco em Java e Spring Boot, atuando na construção de APIs REST escaláveis, seguras e bem estruturadas. Possuo experiência no desenvolvimento de aplicações utilizando Java , Spring Boot e MySQL , aplicando arquitetura em camadas, integração com banco de dados e boas práticas como tratamento global de exceções e organização orientada à manutenção e escalabilidade. Como projeto prático, desenvolvo uma API de miniblog com autenticação e autorização via JWT, incluindo funcionalidades de gestão de usuários e posts, organização por tags e padronização de respostas da aplicação. Também possuo experiência complementar em desenvolvimento Full Stack, utilizando JavaScript, Node.js e React para criação de interfaces responsivas e integração com APIs. Atualmente, curso Análise e Desenvolvimento de Sistemas e continuo evoluindo em conceitos como estrutura de dados, algoritmos e engenharia de software, com foco em escrever código limpo, reutilizável e orientado a boas práticas.
