Desenvolvedor backend em formação com foco em Java e Spring Boot, atuando na construção de APIs REST escaláveis, seguras e bem estruturadas. Possuo experiência no desenvolvimento de aplicações utilizando Java , Spring Boot e MySQL , aplicando arquitetura em camadas, integração com banco de dados e boas práticas como tratamento global de exceções e organização orientada à manutenção e escalabilidade. Finanças Pro - Controle Financeiro Pessoal

Desenvolvi uma aplicação web full stack para gerenciamento de finanças pessoais, com backend em Spring Boot e frontend em HTML/CSS/JS puro. O sistema oferece cadastro de usuários, login com autenticação JWT, CRUD completo de transações (receitas/despesas), resumo financeiro com saldo atualizado, gráfico de pizza interativo para visualizar gastos por categoria e tema claro/escuro. O deploy foi realizado no Render (backend) e Vercel (frontend), garantindo escalabilidade e disponibilidade. O projeto aplica conceitos de segurança, isolamento de dados por usuário, consumo de APIs REST, responsividade e boas práticas de desenvolvimento.
